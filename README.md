# Fitness Tracker

## Overview

Fitness Tracker is a frontend web application built with **React**, **Tailwind CSS**, and **React Router**.  
It allows users to:

- Log their workouts (exercise, sets, reps, weight)
- View workout history with timestamps
- Track total workouts, total weight lifted, and average reps
- Browse exercises fetched from the [WGER API](https://wger.de/en/software/api)

This project simulates a real-world frontend development workflow, including API integration, state management, and responsive UI design.

---

## Features

### Dashboard

- Log workouts with a form
- View workout history
- Summary cards:
  - Total Workouts
  - Total Weight Lifted
  - Average Reps

### Exercises

- Displays exercises from WGER API
- Shows exercise name and description
- Responsive grid layout

### Navigation

- Navbar with links to Dashboard and Exercises pages
- Landing Page with “Get Started” button

---

## Technologies Used

- **React** – Component-based UI
- **Tailwind CSS** – Utility-first styling
- **React Router** – Page navigation
- **Fetch API** – Fetch exercises from WGER API
- **Vite** – Project bundler and dev server
